# 文件命名规范

## 讨论产出

```
discussions/{YYYYMMDD}_{主题}/{YYYYMMDD-HHMM}_{主题}_{视角名}.md
```

- 目录按日期+主题组织
- 文件精确到分钟，主题聚合同一轮，视角名区分各agent

## 会饮记录

```
discussions/{YYYYMMDD}_{主题}/{YYYYMMDD-HHMM}_{主题}_会饮录.md
```

## 示例

```
discussions/20260411_职业选择/
├── 20260411-1430_职业选择_道家.md
├── 20260411-1430_职业选择_荣格.md
├── 20260411-1430_职业选择_加缪.md
└── 20260411-1432_职业选择_会饮录.md
```
